Google Earth toolbox contents

googleearth

With this function, you can automatically add the right subdirectories to the path, update the documentation style sheet with your text colors, and set the location of your system browser.

Syntax

googleearth -docinstall
googleearth -setbrowser
googleearth -version

Description

googleearth -docinstall

After adding the toolbox location to the path, you should run this command. It adds a few subdirectories to the path, and updates the colors of the CSS style sheet such that the GoogleEarth Toolbox documentation uses your text color preferences for strings, keywords, etc. (If you don't use this command, you should still be able to use the documentation). Note that this command is necessary to get the right version number when issuing a googleearth -version command.

googleearth -setbrowser

This option has been added to overcome a problem with using the system browser from the MATLAB documentation (for instance when opening external web pages). Running the command with this option lets you specify the call to launch a browser on your system. This call is then used by the GoogleEarth Toolbox documentation whenever an external web page is requested.

Examples:

"C:\\Program Files\\Internet Explorer\\iexplore.exe" %s
"C:\\Program Files\\Mozilla Firefox\\firefox.exe" %s

Note the double quotes around the location of your browser executable, and the double backslashes. The %s will be substituted by the webaddress.

googleearth -version

This option returns the version number of the toolbox.
